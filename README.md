# Сайт-тренажёр по русскому языку

Статический учебный сайт для тренировки заданий по русскому языку. Готовые темы: фразеологизмы и ударения.

## Структура

- `index.html` — главная страница со списком тренажёров.
- `frazeologizmy/index.html` — раздел «Фразеологизмы».
- `udareniya/index.html` — раздел «Ударения» со словником, тенденциями и тренировочными режимами.
- `.nojekyll` — служебный файл для публикации на GitHub Pages без обработки Jekyll.
