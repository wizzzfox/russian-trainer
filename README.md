# Сайт-тренажёр по русскому языку

Статический учебный сайт для тренировки заданий по русскому языку. Готовые темы: фразеологизмы и ударения.

## Структура

- `index.html` — главная страница со списком тренажёров.
- `frazeologizmy/index.html` — раздел «Фразеологизмы».
- `udareniya/index.html` — раздел «Ударения» со словником, тенденциями, карточками и режимом «ЕГЭ: задание 4».
- `.nojekyll` — служебный файл для публикации на GitHub Pages без обработки Jekyll.

## Публикация на GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить в него содержимое папки `D:\Сайт-тренажёр`.
3. В настройках репозитория открыть `Pages`.
4. Выбрать публикацию из ветки `main`, папка `/root`.
5. Дождаться появления адреса сайта.

Сайт не использует базу данных, регистрацию, личные кабинеты и сборку. Для публикации достаточно этих файлов.
